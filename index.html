Pasos para un algoritmo sencillo secuencial

Sumar dos numeros

  1. Importar la clase java.util
  2. Crear las variables de tipo int o double: numero1, numero2 y resultado
  3. Crear una variable para ingresar los numeros, la cual se le asignara al objeto Scanner
  4. Solicitar al usuario el numero 1 y asignarlo a la variable numero1
  5. Solicitar al usuario el numero 2 y asignarlo a la variable numero2
  6. Sumar los dos numeros y asignar el resultado a la variable resultado
  7. Imprimir la variable resultado con System.out.print
